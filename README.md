# lovable

AI app-builder playground with Prisma, Docker, and sandboxed code execution.

## Stack

Prisma, Docker, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/lovable.git
cd lovable
bun install
bun run dev   # or: bun run start
```
